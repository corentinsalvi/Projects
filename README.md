Bitcoin Price Prediction - ML Project

General Description: 
This project implements a machine learning system to predict Bitcoin prices using 15-minute interval data from 2018 to 2024.
The goal is to forecast Bitcoin prices 1 hour ahead and test trading strategy.

Main Features:
- Price Prediction: Forecast Bitcoin prices 1 hour in advance
- Technical Analysis: Comprehensive feature engineering with financial indicators
- Visual Analytics: Complete visualization suite for results interpretation
- Trading Strategy: Automated trading simulation with performance analysis
- Model Evaluation: 8 different performance metrics for assessment

Languages Used: 
- Python
- Libraries: pandas, numpy, scikit-learn, matplotlib, seaborn
